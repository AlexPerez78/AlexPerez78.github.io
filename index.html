Alarm Notification for Technician (A.N.T)

Sponsored By AT&T
Android App (Completed May 2017)

A.N.T. is a native Android application that was built for At&t/Direct TV technicians for my senior design project. We were instructed to build an application where technicians can relay on a mobile system to recieve phone notifications on which ground station would be down. This was used with several programs such as Jersey and Mongo in order to recieve real upstream data that these sattelties would disperse.
The main goal of the application was to offer a better solution on how quick a technicians can address a problem with the station as more time passes by would create angry customers.
